The Oxygen font family is a desktop/gui font family for integrated use with the KDE desktop and other linux/freetype based graphical user desktops and OS's. The webfont versions of Oxygen Fonts are designed to be used freely across the internet by web browsers on desktop computers, laptops and mobile devices.
All source files for the Oxygen Font family are hosted at https://projects.kde.org/projects/playground/artwork/oxygen-fonts
